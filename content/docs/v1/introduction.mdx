# Introduction

Meet Docxes the next level documentation framework for nextjs. It focuses on Simplicity, high customization across User Interface and zero core package.

<MdxAlert title="Information" type="info">
  Just Clone it and use it with zero core package and auto generation sidebar content.
</MdxAlert>

## What's Initialized

- Enhanced component library
- Improved search performance
- Better dark mode support
- Extended MDX capabilities

## Getting Started

The sidebar on the left will help you navigate through the documentation. Each page is automatically generated from markdown files in the `content/docs` directory.

You can create new pages by adding `.md` or `.mdx` files to the appropriate folders.
