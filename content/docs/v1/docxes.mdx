**DocXes** is a next-level documentation framework built for developers who demand full control without the overhead of proprietary core packages. It leverages the cutting-edge performance of **Next.js 16 (Turbopack)** and **React 19** to deliver a seamless, file-system driven documentation experience.

<MdxAlert title="The Zero-Core Philosophy" type="info">
  DocXes is not a black box. There are no compiled "core" npm packages hiding the logic. You own the code, you control the framework, and you customize every pixel.
</MdxAlert>
