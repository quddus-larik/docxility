These components are automatically imported and available in all MDX files.

## `MdxAlert`

Renders a colored alert box for emphasis.

### Props

<MdxTable 
  headers={["Prop", "Type", "Default", "Description"]}
  rows={[
    ["`type`", '`"info" | "warning" | "success" | "error"`', '`"info"`', "The semantic style of the alert."],
    ["`title`", "`string`", "-", "The bold title at the top of the alert."],
    ["`children`", "`ReactNode`", "-", "The content of the alert."]
  ]}
/>

### Example

```jsx
<MdxAlert type="warning" title="Watch out!">
  This is a warning message.
</MdxAlert>
```

## `MdxCard`

Renders a clickable card, useful for navigation grids.

### Props

<MdxTable 
  headers={["Prop", "Type", "Default", "Description"]}
  rows={[
    ["`title`", "`string`", "-", "The title of the card."],
    ["`href`", "`string`", "-", "The URL to link to."],
    ["`variant`", '`"default" | "secondary" | "outline"`', '`"default"`', "Visual style variant."],
    ["`children`", "`ReactNode`", "-", "The body text of the card."]
  ]}
/>

### Example

```jsx
<MdxCard title="Read the Guide" href="/docs/guide" variant="secondary">
  Click here to learn more.
</MdxCard>
```

## `StepsWithCounter` / `Step`

Renders a vertical list of steps with a connecting line.

### Props (StepsWithCounter)

<MdxTable 
  headers={["Prop", "Type", "Default", "Description"]}
  rows={[
    ["`children`", "`ReactNode`", "-", "Should contain `Step` components."]
  ]}
/>

### Props (Step)

<MdxTable 
  headers={["Prop", "Type", "Default", "Description"]}
  rows={[
    ["`children`", "`ReactNode`", "-", "The content of the step. Markdown is supported."]
  ]}
/>

### Example

```jsx
<StepsWithCounter>
  <Step>
    #### First Action
    Do this thing first.
  </Step>
  <Step>
    #### Second Action
    Then do this.
  </Step>
</StepsWithCounter>
```
