# Theming

DocXes provides a multi-layered theming system. You can switch between "Marketplace" formulas, tweak global CSS variables, or provide granular style overrides.

## The Marketplace

The `marketplace/` directory contains pre-built UI formulas. Each formula defines a unique aesthetic:

- **Shadcn UI**: High-contrast, professional, minimalist.
- **Curved UI**: Soft edges, elevated surfaces, modern pill-shaped elements.
- **Minimal UI**: Ultra-clean, focused on typography, borderless.

To apply a theme, spread it into your configuration:

```tsx
import { MinimalUI } from "@/marketplace/minimal-ui";

export const XMeta = createConfig({
  ...MinimalUI,
  // Your overrides go here
});
```

## CSS Variables (`cssVars`)

Our **Headless Tailwind** system allows you to redefine your site's color palette and geometry in `x-meta.config.tsx`. We use OKLCH colors for better perceptual consistency.

```tsx
theme: {
  cssVars: {
    root: {
      "radius": "1rem", // Global border radius
    },
    light: {
      "primary": "oklch(0.6 0.2 260)", // Your brand color
      "background": "oklch(0.99 0 0)",
    },
    dark: {
      "primary": "oklch(0.8 0.15 260)",
      "background": "oklch(0.12 0 0)",
    }
  }
}
```

## Granular Style Overrides

If a theme is 90% perfect but you want to tweak specific parts, use the `styles` object. This allows you to append or replace Tailwind classes for core components.

```tsx
sidebar: {
  styles: {
    sidebar: "bg-muted/50 border-r", // Sidebar container
    item: "rounded-lg px-4 py-2",    // Nav links
    itemActive: "bg-primary/20",     // Active state
  }
},
toc: {
  styles: {
    title: "text-primary font-bold", // "On this page" title
  }
}
```

## Syntax Highlighting

You can customize the code block experience via the `theme.mdx` config:

```tsx
theme: {
  mdx: {
    highlighter: "pretty-code",
    theme: "nord", // Any Shiki compatible theme
    keepBackground: false, // Forces code blocks to inherit container bg
  }
}
```

<MdxCard title="Pro Tip" variant="secondary">
  When `keepBackground` is set to `false`, DocXes will force the code block background to be transparent, allowing it to blend perfectly with your container's `bg-muted` or `bg-background` styles.
</MdxCard>
